    5.2.2 闲置信息模块的设计与实现
    有效用户可以首页浏览闲置信息，在首页的闲置信息模块可以浏览系统中最新发布的闲置信息列表。有效用户可以在首页查看本系统的数据统计，包括有今日通告、总注册人数、总发布数量、总成交量。用户通过查看通告栏可以快速的了解校园动态。用户可以通过点击相应的列表进入闲置信息详情浏览闲置物品的信息。在物品的详情页中，如是用户本人发布的物品则显示“出示二维码”和“删除”按钮。
